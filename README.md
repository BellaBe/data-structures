This is an implementation of data structures in Javascript(ES6)

Linked list supports:
-getListLength()
-addFirst()
-addLast()
-removeFirst()
-removeLast()
-insertAt()
-findIndexOfElement()
-elementAtIndex()
-listToArray()
-isEmpty()

Stack supports:
-push()
-pop()
-getStackLength()
-getLast()
-printStack()
-isEmpty()
